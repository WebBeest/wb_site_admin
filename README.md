# Web-Beest - Site administration
This recipe is designed to configure a list of modules and configuration needed
for a nice site administration experience in a Drupal site.

## Features
Adds and configures the following modules:
* [Admin toolbar](https://drupal.org/project/admin_toolbar)
* [Configuration ignore](https://drupal.org/project/config_ignore)
* [Configuration split](https://drupal.org/project/config_split)
* [Masquerade](https://drupal.org/project/masquerade)
* [Rabbit hole](https://drupal.org/project/rabbit_hole)
